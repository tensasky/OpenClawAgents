## 核心功能

### 数据采集
- 全A股 5,348只
- minute实时数据 (每5分钟)
- 日线数据 (每日更新)
- 进程锁防冲突

### 自动交易
- 红中信号 → 自动买入
- 涨停过滤 (≥9.9%禁买)
- 滑点计算 (0.2%)
- 移动止损止盈

### 通知系统
- Discord webhook
- Email SMTP
- 统一模板

## 快速开始

```bash
# 测试所有Agent
bash test_all_agents.sh

# 采集数据
python3 beifeng/beifeng.py <股票> --type daily|minute

# 生成信号
python3 hongzhong/generate_signals_v3.py

# 自动交易
python3 facai/facai.py

# 监控
python3 scripts/caishen_monitor_v52.py
```

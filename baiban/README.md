# 白板 (The Whiteboard) —— 策略进化与回测系统

🀆 **职责**：收盘后/周末全量回测，策略进化，动态调整参数

## 核心定位

白板拥有"上帝视角"。不参与实时决策，但在每天收盘后及每周周末执行全量回测。

**核心问题**："如果按照这套参数跑过去三年，我会亏死还是翻倍？"

## 核心功能

### 1. 归因分析 (Attribution Analysis)

拉取发财的所有交易记录，拆解买入理由：

- **胜率拆解**：西风热点板块 vs 南风技术金叉，哪个赢得多？
- **失败复盘**：记录所有触发止损的单子，分析是大盘环境不好还是指标滞后

### 2. 市场情绪感知器 (Market Sentiment Sensor)

| 市场情绪 | 判定标准 | 白板动态建议 |
|----------|----------|--------------|
| **亢奋 (Bull)** | 大盘MA20向上，上涨家数>3000 | 进攻型：题材权重+20%，止盈位15%+，放宽RSI |
| **震荡 (Flat)** | 大盘BOLL中轨，成交量持平 | 波段型：严格收盘前15分钟买入，止盈5%，增强BOLL权重 |
| **恐慌 (Bear)** | 大盘跌破MA60，跌停家数激增 | 防守型：初始分-2，止损3%，非High热点不准入池 |

### 3. 动态收益率优化

- **动态止盈优化**：如果最近10笔盈利中有8笔卖出后继续涨5%，通知发财将止盈步长从1%提高到2.5%
- **打分权重自适应**：如果连续三周技术分不如题材分，自动修改南风公式：
  ```
  Score = (Theme × 0.6) + (Tech × 0.4)
  ```

### 4. 自动化报告与进化指令

每周生成《策略进化清单》，下发给各Agent：

```
📈 [白板·策略进化指令]
● 本周回测表现：胜率58%，回撤2.1%，资产增值4.5%
● 情绪诊断：当前处于【震荡转向恐慌】阶段
● 进化调整：
  - 南风：将"大盘破位"权重提高1.5倍
  - 东风：活跃度门槛从3%提高到4%
  - 发财：强制降低单笔仓位至20%
[下次回测：周日20:00]
```

## 文件结构

```
baiban/
├── baiban.py              # 主程序
├── backtest.py            # 回测引擎
├── attribution.py         # 归因分析
├── sentiment.py           # 情绪感知器
├── optimizer.py           # 参数优化
├── data/
│   ├── trades_history.json    # 发财交易记录
│   ├── backtest_results.json  # 回测结果
│   ├── sentiment_state.json   # 当前情绪状态
│   └── evolution_log.json     # 进化历史
├── reports/               # 周报目录
├── logs/                  # 日志目录
└── README.md              # 本文件
```

## 运行方式

```bash
# 每日收盘后回测（15:30后）
python baiban.py --daily

# 周末全量回测（周日20:00）
python baiban.py --weekly

# 归因分析
python baiban.py --attribution

# 情绪诊断
python baiban.py --sentiment

# 生成进化指令
python baiban.py --evolve

# 查看回测报告
python baiban.py --report
```

## 七Agent闭环

```
🌪️ 北风(数据) → 🍃 西风(舆情) → 🌬️ 南风(大脑)
                              ↓
🌸 东风(漏斗) → 🀄 红中(哨兵) → 💰 发财(实战)
                              ↓
                        🀆 白板(复盘/进化)
                              ↓
                        [参数调整] → 南风/东风/发财
```

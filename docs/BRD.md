## 1. 项目概述

### 1.1 项目背景
A股市场每天产生海量数据，个人投资者面临信息不对称、反应滞后等痛点。本系统通过9-Agent协同架构，实现从数据采集到自动交易的完整闭环。

### 1.2 目标用户
- 散户投资者
- 量化交易爱好者
- 需要自动化交易系统的用户

### 1.3 核心价值
- **效率提升**: 5分钟数据采集 → 实时信号响应
- **决策辅助**: 多因子评分 + ATR动态止损
- **风险控制**: 行业30%限制 + 每日5笔限额

---

## 2. 市场需求

### 2.1 功能需求
| 优先级 | 功能 | 描述 |
|--------|------|------|
| P0 | 数据采集 | 全A股 5,348只 minute+daily |
| P0 | 信号生成 | 评分≥65分触发买入信号 |
| P0 | 自动交易 | 信号→买入，涨停过滤 |
| P1 | ATR动态止损 | 根据波动率自动调整 |
| P1 | 组合风控 | 行业限制+每日限额 |
| P2 | 事件驱动 | Redis Pub/Sub秒级响应 |

### 2.2 性能需求
- 数据采集延迟: < 5分钟
- 信号响应时间: < 10秒
- 系统可用性: 99%

### 2.3 成本约束
- 初始资金: 10万/策略
- 手续费: 万分之3
- 滑点: 0.2%

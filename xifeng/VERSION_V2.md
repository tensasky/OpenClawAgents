# 西风 V2.0 版本说明

**版本号**: V2.0.0  
**发布日期**: 2026-03-13  
**升级内容**: 板块分析加强 + 定时Discord推送

---

## 🎯 版本特性

### 1. 板块分析加强 ✅

**功能**:
- 实时板块热度排名
- 板块涨跌幅统计
- 龙头股自动识别
- 市场情绪分析

**数据来源**:
- 主版本: akshare实时板块数据
- 备用版本: 本地数据库分析

### 2. 定时Discord推送 ✅

**推送频率**: 每2小时
**推送内容**:
- 热门板块Top10
- 市场情绪指标
- 龙头股表现
- 投资建议

**推送时间**:
- 09:30 (开盘)
- 11:30 (上午收盘)
- 13:30 (下午开盘)
- 15:30 (下午收盘)
-  etc.

## 🚀 部署方式

### 方式1: 完整版（推荐）

使用akshare获取实时板块数据：

```bash
python3 xifeng_v2_sector.py
```

**优点**: 数据实时、板块分类准确
**缺点**: 依赖网络，可能受限制

### 方式2: 简化版

使用本地数据库分析：

```bash
python3 xifeng_v2_simple.py
```

**优点**: 不依赖外部接口，稳定
**缺点**: 板块分类较简单

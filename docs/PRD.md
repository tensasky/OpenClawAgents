# A股智能交易系统 - 产品需求文档 (PRD)

**版本**: v4.8.1  
**日期**: 2026-03-19

---

## 1. 产品概述

A股量化交易系统，基于9-Agent协同架构，实现全自动选股、交易、风控。

---

## 2. 系统架构

### 2.1 Agent矩阵

| Agent | 功能 | 版本 |
|-------|------|------|
| 北风 | 数据采集 | V3.1 |
| 西风 | 热点板块 | V2.0 |
| 东风 | 候选池 | V1.0 |
| 南风 | 策略评分 | V5.5 |

## 3. 风险管理

| 功能 | 描述 |
|------|------|
| ATR动态止损 | StopLoss = Highest - k×ATR |
| 板块相关性 | 集群限制40% |
| T+1限制 | is_sellable字段 |
| 跌停预警 | 流动性监控 |

---

## 4. 交易规则

| 规则 | 参数 |
|------|------|
| 初始资金 | 10万/策略 |
| 单股仓位 | 最大50% |
| 滑点 | 买入0.2%, 卖出0.2% |
| 手续费 | 万分之3 |
| T+1 | 当日买入次日可卖 |

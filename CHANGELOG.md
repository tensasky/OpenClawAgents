## [v1.3.0] - 2026-03-10

### 新增
- **北风准实时数据机制**
  - 交易时段：分钟数据实时聚合成日线
  - 收盘后：自动补充完整日线数据
  - 任何Agent调用：始终返回最新数据
  - 数据延迟：1-5分钟

- **北风数据优化（接近交易所级别）**
  - 智能缓存：内存+本地5分钟缓存，减少90%重复请求
  - 请求限流：动态间隔30-120秒，防封IP
  - 多源备份：腾讯/新浪自动切换
  - 增量更新：只抓缺失数据段
  - 实时采集：每30秒轮询活跃股票

- beifeng_realtime_collector.py - 实时采集器（1-5分钟延迟）

- core_monitor.py - 核心监控脚本

## [v1.0.0] - 2026-03-09

### 新增
- **北风 (BeiFeng) v1.0** - 股票数据采集 Agent
  - 支持 2,470 只 A 股日线数据
  - 多数据源（腾讯财经、新浪财经）
  - 自动补全历史数据

  - 分级输出（High/Medium/Low）
  - 定时分析（每30分钟）
  - hot_spots.json 输出

  - 基础框架
  - 代码审查接口
  - 代码生成功能

- **监控 (Monitor) v0.1** - 系统监控 Agent
  - Agent 状态检查
  - 健康报告生成

## 1. 项目概述

### 1.1 项目背景
A股市场每天产生海量数据，个人投资者面临信息不对称、反应滞后等痛点。本系统通过9-Agent协同架构，实现从数据采集到自动交易的完整闭环。

### 1.2 核心价值
- **效率提升**: 5分钟数据采集 → 实时信号响应
- **决策辅助**: 多因子评分 + ATR动态止损
- **风险控制**: 行业30%限制 + 板块相关性 + T+1

## 3. 核心架构

### 3.1 数据流
```
数据源 → 北风 → 判官 → 数据库
              ↓
西风(热点) → 东风(筛选) → 南风(评分) → Redis Streams → 发财(交易)
                                              ↓
                                      哨兵(监控+预警)
```

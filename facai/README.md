# 发财 (The Trader) - 模拟交易与资产管理专家

💰 **职责**：模拟交易执行、动态风控、资产管理

## 账户概览

- **初始资金**：100,000.00 CNY
- **交易标的**：南风打分 Top 3 且红中已预警的股票
- **持仓上限**：单只股票不超过 30%-50%

## 核心功能

### 1. 买入执行 (Entry Strategy)
- **时间窗**：14:50 - 15:00（收盘前最后10分钟）
- **依据**：南风打分 ≥ 8.5 且东风记录为"强板块效应"
- **买入理由**：自动记录南风技术分 + 西风热点等级

### 2. 动态风控算法

#### 动态止损 (Trailing Stop Loss)
- 初始止损：买入价 -5%
- 随股价上涨平移（涨10% → 止损上移至+5%）

#### 情绪止盈
- 西风热点评分从 "High" 掉到 "Low" → 强制减仓/清仓

#### 波动止损
- 结合北风 ATR 指标
- 当日振幅异常放大且向下 → 提前清仓

### 3. 模拟账户管理

数据库：`portfolio.db`

| 字段 | 内容 | 示例 |
|------|------|------|
| timestamp | 交易时间 | 2026-03-05 14:55:02 |
| action | 买入/卖出 | BUY |
| symbol | 股票代码 | sh600XXX |
| price | 成交均价 | 15.30 |
| quantity | 股数 | 2000 |
| logic | 买卖理由 | 尾盘抢筹+西风高热点；技术面回踩5日线支撑 |
| total_assets | 总资产 | 102,450.00 |

### 4. 交易后通知
- 交易确认后1秒内触发
- 多渠道：Discord + 飞书
- 包含：成交信息、理由、持仓、风控策略

## 运行方式

```bash
# 检查并执行买入（14:50-15:00）
python facai.py --buy

# 检查风控条件（持续监控）
python facai.py --monitor-risk

# 查看账户概览
python facai.py --portfolio

# 查看交易历史
python facai.py --history
```

## Agent 协作

```
红中预警 (14:45)
      ↓
发财获取 Top 3 名单
      ↓
14:50-15:00 执行买入
      ↓
portfolio.db 记录
      ↓
动态风控持续监控
      ↓
触发条件 → 卖出 + 通知
```

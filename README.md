```

        ↑___________________________↓
              财神爷(监督协调)
```

## 📊 V3.3.0 核心特性

### 1. 红中V3.3 - Discord表格格式通知 ✅
- **两块展示**: 🛡️保守策略 + ⚖️平衡策略
- **8列信息**: 代码/名称/价格/策略/评分/板块/趋势/热点/消息
- **真实数据**: 所有价格基于北风实时数据
- **清晰对齐**: 代码块表格样式，易于阅读

### 2. 双版本策略
- **保守版**: 胜率85%，收益8-13%，15笔/月
- **平衡版**: 胜率65%，收益5-8%，75笔/月

### 3. 数据库物理隔离
- **stocks_real.db**: 真实历史数据（日线+分钟）
- **stocks_virtual.db**: 实时虚拟数据（虚拟日线）
- **已清理**: 删除旧stocks.db，统一使用新数据库

### 4. 高频监控
- 北风: 交易时段每5分钟采集
- 西风: 每2小时板块分析
- 红中: 每30分钟扫描预警
- 财神爷: 每小时状态报告

**作者**: Roberto & AI Agent

---

## 1. 产品概述

### 1.1 产品定位
A股智能量化交易系统，基于9-Agent协同架构，实现全自动化的选股、交易、风控流程。

### 1.2 核心价值
- **全自动化**: 从选股到交易全流程无需人工干预
- **数据驱动**: 基于实时数据和策略评分
- **风险控制**: 移动止损止盈、涨停过滤
- **多维度选股**: 宏观(板块) + 微观(个股)联动

## 2. 系统架构

### 2.1 9-Agent协同架构

```
┌─────────────────────────────────────────────────────────┐
│                    财神爷 (监控)                        │
│                   监督整个系统运行                        │
└─────────────────────────────────────────────────────────┘
                              │
    ┌─────────────┬─────────────┴─────────────┐
    │             │                           │
┌───▼────┐  ┌──▼────┐  ┌────────────▼────────┐
│  西风   │  │  东风  │  │       南风        │
│ 热点板块 │  │候选股池│  │     策略评分      │
└────┬────┘  └───┬────┘  └────────┬───────────┘
     │            │                  │
     │            │                  ▼
     │            │         ┌───────────────┐
     │            │         │    红中      │
     │            │         │   交易信号   │
     │            │         └───────┬───────┘
     │            │                 │
     ▼            ▼                 ▼
┌──────────────────────────────────────┐
│              发财 (模拟交易)           │
│         自动买入/卖出/风控            │
└──────────────────────────────────────┘
```

### 2.2 数据流

```
数据源 → 北风(采集) → 判官(校验) → 数据库
                              ↓
西风(热点) → 东风(筛选) → 南风(评分) → 红中(信号) → 发财(交易)
```

## 3. 功能模块

### 3.1 Agent详情

| Agent | 功能 | 版本 | 核心能力 |
|-------|------|------|---------|
| 北风 | 数据采集 | V3.1 | minute+daily, 进程锁 |
| 西风 | 热点板块 | V2.0 | 板块识别, 热度评分 |
| 东风 | 候选池 | V1.0 | 量比筛选, 振幅筛选 |
| 南风 | 策略评分 | V5.5 | 多策略评分 |
| 红中 | 交易信号 | V3.4 | 信号生成, 通知 |
| 发财 | 模拟交易 | V1.1 | 自动买入, 止损止盈 |
| 判官 | 数据校验 | V1.2 | 数据聚合, 质量校验 |
| 白板 | 策略优化 | V1.0 | 收盘复盘 |
| 财神爷 | 系统监控 | V5.2 | 全量监控 |

### 3.2 联动选股系统

**工作流**:
1. 西风识别热点板块
2. 东风筛选板块内活跃股 (量比>2, 振幅>3%)
3. 南风策略评分
4. 红中生成交易信号
5. 发财执行交易

### 3.3 交易规则

| 规则 | 参数 |
|------|------|
| 初始资金 | 10万/策略 |
| 单股仓位 | 最大50% |
| 滑点 | 买入0.2%, 卖出0.2% |
| 手续费 | 万分之3 |
| 止损 | 初始-5%, 移动止损 |
| 涨停过滤 | ≥9.9%禁止买入 |

## 6. 接口定义

### 6.1 数据接口

```python
# 获取股票信息
get_stock_detail(code: str) -> dict

# 获取实时价格
get_realtime_price(code: str) -> float

# 获取板块热点
get_hot_sectors() -> list
```

### 6.2 交易接口

```python
# 买入
buy(symbol, price, quantity) -> bool

# 卖出
sell(symbol, price, reason) -> bool

# 获取持仓
get_positions() -> list
```

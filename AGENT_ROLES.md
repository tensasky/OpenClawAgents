# Agent 分工说明

## 南风 vs 红中 职责划分

### 南风 (NanFeng) - 量化打分引擎
**职责**: 专注股票评分和筛选
- 技术指标计算 (ADX, RSI, MACD, MA等)
- 多维度打分 (趋势40% + 动量30% + 成交量20% + 质量10%)
- 实时数据聚合 (分钟数据→日线)
- 输出: 股票代码 + 分数 + 技术指标

**不处理**:
- 股票中文名称查询
- 消息通知发送
- 定时任务调度

### 红中 (HongZhong) - 预警通知专家
**职责**: 交易时段实时计算+通知
- 定时扫描 (交易时段每N分钟或每日14:45)
- 调用南风打分
- 查询股票中文名称
- 格式化预警消息
- 多渠道推送 (Discord/飞书)

**工作流程**:
```
交易时段 → 红中定时触发 → 调用南风打分 → 查询股票名称 
→ 格式化消息 → 推送通知
```

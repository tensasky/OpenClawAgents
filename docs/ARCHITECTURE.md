# OpenClaw 量化交易系统架构文档

## 1. 系统概述

**系统名称**: OpenClaw A股智能交易系统  
**版本**: v4.3.0  
**更新日期**: 2026-03-27

### 核心能力
- 9-Agent协同工作
- 5分钟快速选股链路
- 全自动化交易闭环
- 实时数据监控

---

## 2. Agent架构

| Agent | 职责 | 关键文件 |
|-------|------|----------|
| **北风** | 数据采集 | beifeng.py |
| **西风** | 板块分析 | xifeng/*.py |
| **东风** | 候选池筛选 | dongfeng_v2.py |
| **南风** | 策略评分 | strategy_db.py |
| **红中** | 信号生成 | quick_scan.py |
| **发财** | 交易执行 | facai/*.py |
| **白板** | 回测优化 | baiban/*.py |
| **判官** | 数据校验 | judge/*.py |
| **财神爷** | 调度管理 | manager/*.py |

## 6. 核心配置

### 策略参数
```json
{
  "filters": {
    "min_ma20_slope": 0.002,
    "min_rsi": 40,
    "max_rsi": 75
  },
  "weights": {
    "bullish": 30,
    "ma20_slope": 20,
    "rsi": 20
  }
}
```

### 风控参数
- 资金: ¥100,000
- 单只上限: ¥10,000
- 持仓上限: 10只
- 仓位上限: 80%
- 滑点限制: 1.5%

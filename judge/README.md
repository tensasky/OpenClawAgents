# ⚖️ 判官 (Judge) - 数据验证Agent

**版本**: V1.0.0  
**职责**: 验证北风数据准确性，确保数据质量  
**Emoji**: ⚖️

---

## 🎯 核心职责

判官是数据质量的守护者，负责：

1. **验证数据覆盖率** - 确保所有股票数据完整
2. **验证价格准确性** - 对比新浪等权威数据源
3. **验证数据一致性** - 日线和分钟数据匹配
4. **生成验证报告** - 输出数据质量报告

---

## 📊 验证维度

| 检查项 | 阈值 | 说明 |
|--------|------|------|
| 日线覆盖率 | ≥5000只 | 每日至少5000只股票有日线数据 |
| 分钟覆盖率 | ≥5000只 | 每日至少5000只股票有分钟数据 |
| 价格准确性 | <1%误差 | 与新浪数据差异不超过1% |
| 数据一致性 | 0处不一致 | 日线和分钟收盘价一致 |

---

## 🚀 使用方法

```bash
# 运行判官验证
python3 judge/judge_agent.py

# 查看报告
ls judge/reports/judge_report_*.json
```

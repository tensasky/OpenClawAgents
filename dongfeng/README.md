# 东风 - 股票初筛与池管理 Agent

🌸 **职责**：从北风数据源初筛活跃股票，管理股票池和备用池

## 核心功能

### 1. 股票初筛
- 读取北风数据库的日线/分钟线数据
- 筛选条件：
  - 振幅 > 3%（当日最高价-最低价/昨收）
  - 温和放量（成交量较前5日均量增长 20%-100%）
  - 排除 ST、退市风险股

### 2. 股票池管理
- **活跃池**：符合筛选条件的股票
- **备用池**：不再活跃的股票，记录退出原因

### 3. 数据记录
- 进入股票池的时间
- 退出股票池的时间
- 退出原因（振幅不足/放量异常/其他）

## 运行方式

```bash
# 手动运行筛选
python dongfeng.py --scan

# 查看当前股票池
python dongfeng.py --list

# 查看备用池
python dongfeng.py --backup
```

## 与北风的关系

```

```

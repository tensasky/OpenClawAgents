# 东风 - 股票初筛与池管理 Agent (实时交易版)

🌸 **职责**：交易时段实时监控，初筛活跃股票，管理候选池和备用池

## 核心触发机制

- **监控窗口**：交易日 13:30 - 14:15（收盘前 45-90 分钟）
- **轮询频率**：每 15 分钟执行一次扫描
- **协同关系**：
  - 读取西风的 `hot_spots.json`（锁定热门行业）
  - 读取北风的实时/准实时行情
  - 输出至候选池，供南风进行深度打分

## 筛选逻辑

### 1. 振幅 (Amplitude)
当日 (最高 - 最低) / 昨收 > 3%
- 代表该股有资金在博弈，不是死水潭

### 2. 温和放量 (Moderate Volume)
当日成交量 / 过去5日平均成交量 介于 **1.2 到 2.5** 之间
- 注：暴增（>5倍）可能是拉高出货或巨大利空，温和增长才是资金建仓的特征

### 3. 题材对齐
优先选择属于西风 **High/Medium** 热点等级的个股

## 双池管理机制

### A. 候选池 (candidate_pool.json) —— 精英突击队
- **进入准则**：满足振幅、放量、题材三项硬指标
- **记录要素**：股票代码、加入时间、加入理由(如：液冷板块+放量1.5倍)、当前振幅

### B. 备用池 (standby_pool.json) —— 待命/淘汰区
- **进入准则**：
  - 指标衰退：曾进入候选池，但下一次轮询时振幅收窄，或成交量萎缩
  - 形态走坏：分时线跌破均价线，或出现急跌信号
  - 热点漂移：西风下调了该股所属板块的热度等级
- **记录要素**：股票代码、淘汰时间、淘汰理由(如：量能枯竭, 涨幅回落)、停留时长

## 运行方式

```bash
# 手动扫描（仅热点股）
python dongfeng.py --scan

# 扫描全部股票
python dongfeng.py --scan --all

# 进入监控模式（13:30-14:15自动扫描）
python dongfeng.py --monitor

# 查看候选池
python dongfeng.py --list

# 查看备用池
python dongfeng.py --standby
```

## Agent 协作流程

```
西风(热点分析) → hot_spots.json
                      ↓
北风(实时数据) → SQLite DB → 东风(初筛选股) → candidate_pool.json
                                                  ↓
                                            南风(深度打分)
```

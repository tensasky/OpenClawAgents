# A股智能交易系统 - 产品需求文档 (PRD)

**版本**: v4.6.1  
**日期**: 2026-03-18

---

## 1. 产品概述

### 1.1 产品定位
A股量化交易系统，基于9-Agent协同架构，实现全自动选股、交易、风控。

### 1.2 核心功能
- 数据采集: minute+daily全量数据
- 信号生成: 多策略评分
- 自动交易: 买入/卖出/止损
- 风险控制: ATR动态止损+组合风控

### 2.2 数据流

```
数据源 → 北风 → 判官 → 数据库
              ↓
西风(热点) → 东风(筛选) → 南风(评分) → 红中(信号) → 发财(交易)
                                              ↓
                                      事件驱动(Redis)
                                              ↓
                                      哨兵(监控)
```

## 3. 功能需求

### 3.1 数据采集
- 全A股 5,348只
- minute: 每5分钟
- daily: 每日更新
- 进程锁防冲突

### 3.2 信号生成
- 评分阈值: ≥65分
- 信号结构:
```python
{
    'code': 'sh600519',
    'name': '贵州茅台',
    'score': 85.0,
    'strategy': '趋势跟踪',
    'signal_id': 123,
    'entry_price': 1500.0,
    'stop_loss': 1425.0
}
```

### 3.3 自动交易
- 买入: 滑点+0.2%
- 卖出: 滑点-0.2% + 手续费0.03%
- 涨停过滤: ≥9.9%禁止买入
- 止损: ATR动态止损

### 3.4 风险控制
- 单股仓位: ≤50%
- 行业限制: ≤30%
- 每日限额: ≤5笔
- 策略资金: 10万/策略

---

## 4. 交易规则

| 规则 | 参数 |
|------|------|
| 初始资金 | 10万/策略 |
| 单股仓位 | 最大50% |
| 滑点 | 买入0.2%, 卖出0.2% |
| 手续费 | 万分之3(仅卖出) |
| 止损 | ATR动态 |
| 涨停过滤 | ≥9.9% |

## 5. 接口定义

### 5.1 信号接口
```python
# 获取信号
def get_signals(strategy: str = None, min_score: int = 65) -> List[Signal]
```

### 5.2 交易接口
```python
# 买入
def buy(signal: Signal) -> bool

# 卖出
def sell(symbol: str, reason: str) -> bool

# 持仓
def get_positions() -> List[Position]
```

### 5.3 风控接口
```python
# 风险检查
def check_buy_risk(signal: Signal) -> (bool, str)

# ATR计算
def calculate_atr(code: str) -> float
```

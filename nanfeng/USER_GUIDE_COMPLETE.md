## 💰 操作流程

### 步骤1: 接收信号 (14:45)
- 关注Discord预警频道或查收邮件
- 查看红中推送的交易信号

### 步骤2: 确认信号 (14:45-14:50)
**检查清单**:
- [ ] 大盘环境: 上证指数ADX>20
- [ ] 板块热度: 股票所在板块为热点
- [ ] 个股状态: 无停牌、无重大利空

### 步骤3: 下单入场 (14:50-15:00)
```
操作: 买入推荐股票
价格: 市价或限价（建议市价确保成交）
数量: 按建议仓位（单票不超过25%）
```

### 步骤4: 设置止损 (入场后立即)
```
条件单设置:
- 条件: 价格跌至止损价
- 操作: 自动卖出全部仓位
- 有效期: 长期有效
```

### 步骤5: 持有跟踪 (每日)
**每日收盘后检查**:
- 当前盈亏比例
- 是否触及目标1/2/3
- 移动止损是否需要调整

### 步骤6: 分批止盈
| 目标 | 涨幅 | 操作 | 卖出比例 |
|------|------|------|---------|
| 目标1 | +6% | 卖出部分 | 30% |
| 目标2 | +12% | 卖出部分 | 40% |
| 目标3 | +20% | 卖出剩余 | 30% |

### 步骤7: 时间止损
- **趋势跟踪**: 15天未达目标，强制离场
- **均值回归**: 5天未达目标，强制离场
- **突破策略**: 7天未达目标，强制离场
- **稳健增长**: 30天未达目标，强制离场
- **热点追击**: 3天未达目标，强制离场

## ⚠️ 风险控制

### 单票限制
- 最大仓位: 不超过总资金25%
- 单策略最大: 不超过总资金30%

### 日度限制
- 每日最多买入: 3只股票
- 每日最大仓位: 不超过60%

### 止损纪律 (必须严格执行)
- ✅ 触及止损立即离场
- ❌ 不允许补仓摊低成本
- ❌ 不允许主观放宽止损

## 📊 5策略详解

### 1. 趋势跟踪
- **特点**: 顺势而为，捕捉中期趋势
- **持有期**: 7-15天
- **胜率**: 保守版85% / 平衡版68%
- **收益**: 保守版10.7% / 平衡版6.5%

### 2. 均值回归
- **特点**: 捕捉超跌反弹
- **持有期**: 2-5天
- **胜率**: 保守版80% / 平衡版57%
- **收益**: 保守版8.1% / 平衡版4.9%
- **注意**: 风险较高，严格止损

### 3. 突破策略
- **特点**: 放量突破关键位置
- **持有期**: 3-7天
- **胜率**: 保守版85% / 平衡版68%
- **收益**: 保守版10.8% / 平衡版7.2%

### 4. 稳健增长
- **特点**: 最高质量筛选，长期持有
- **持有期**: 15-30天
- **胜率**: 保守版85% / 平衡版74%
- **收益**: 保守版8.4% / 平衡版6.5%

### 5. 热点追击
- **特点**: 紧跟市场热点，快进快出
- **持有期**: 1-3天
- **胜率**: 保守版85% / 平衡版72%
- **收益**: 保守版13.1% / 平衡版8.5%
- **注意**: 风险最高，严格止损

## 🆘 紧急情况处理

### 系统故障
1. 联系财神爷排查
2. 手动参考策略条件选股
3. 暂停交易等待恢复

### 市场异常
- 大盘暴跌(>5%): 暂停新信号，持仓视情况止损
- 个股黑天鹅: 立即止损离场
- 板块集体大跌: 检查持仓，必要时减仓

### 个人原因无法操作
- 提前告知财神爷
- 可设置自动交易（需额外配置）
- 或选择保守版减少操作频率

## 🎯 成功关键

1. **严格执行止损**: 单笔亏损控制在2-3%

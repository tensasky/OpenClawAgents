# A股智能交易系统 - 产品需求文档 (PRD)

**版本**: v4.7.0  
**日期**: 2026-03-18

---

## 1. 产品概述

### 1.1 产品定位
A股量化交易系统，基于9-Agent协同架构，实现全自动选股、交易、风控。

### 1.2 核心功能
- 数据采集: minute+daily全量数据
- 信号生成: 多策略评分(6种)
- 自动交易: 买入/卖出/T+1
- 风险控制: ATR+相关性+跌停预警

## 3. 核心功能

### 3.1 多策略信号
1. 趋势跟踪
2. 均值回归
3. 突破策略
4. 稳健增长
5. 涨停监控
6. 热点追击

### 3.2 风险管理
- **ATR动态止损**: StopLoss = Highest - k × ATR
- **板块相关性**: 集群限制40%
- **T+1限制**: is_sellable字段
- **跌停预警**: 流动性监控

### 3.3 事件驱动
- Redis Streams消息队列
- 消息持久化+ACK确认
- 数据库轮询降级

---

## 4. 交易规则

| 规则 | 参数 |
|------|------|
| 初始资金 | 10万/策略 |
| 单股仓位 | 最大50% |
| 滑点 | 买入0.2%, 卖出0.2% |
| 手续费 | 万分之3(仅卖出) |
| T+1 | 当日买入次日可卖 |
| 板块集群 | 最大40% |

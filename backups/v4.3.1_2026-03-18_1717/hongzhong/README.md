# 红中 (Red Dragon) - 决策预警与自动化通信专家

🀄 **职责**：14:45 获取南风Top3，多渠道推送预警

## 核心工作时段

- **运行时间**：14:45
- **任务**：获取南风打分前3名，推送到 Discord + 飞书

## 核心功能

### 1. 获取南风Top3
- 调用南风打分逻辑
- 筛选分数 >= 7.0 的股票
- 按分数排序取前3名

### 2. 多渠道推送
- Discord Webhook
- 飞书 Webhook
- 并发发送

### 3. 重试机制
- 超时：30秒
- 重试：最多3次
- 失败记录日志，不阻塞其他渠道

## 预警消息格式

```
🔔 [财神爷量化预警] #1

📈 **中芯国际** (000938)
⭐ 综合评分: **8.5/10**

💰 价格: ¥45.80
🛑 止损: ¥43.20
🎯 目标: ¥52.50

⏰ 14:45 | 红中🀄
```
